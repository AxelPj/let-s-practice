# Let's Practice

Bienvenue dans mon projet **Let's Practice** ! Ce repository est destiné à héberger mes projets personnels que je réalise pour m'entraîner à coder, principalement en **C**. Actuellement, le projet en cours est un jeu simple en terminal pour pratiquer mes compétences en programmation.

## Projets

### 1. Jeu du Plus ou Moins
Le premier projet que j'ai créé est un jeu du **Plus ou Moins**. Voici comment il fonctionne :

- L'ordinateur choisit un nombre aléatoire entre **1 et 100**.
- Le but du jeu est de deviner ce nombre en un minimum de tentatives.
- Après chaque essai, le programme vous indique si votre tentative est trop grande, trop petite, ou si vous avez trouvé le bon nombre.
